🎨 UI Design Details
FitLife mengusung desain Modern Fitness & Health UI yang bersih, responsif, dan ramah pengguna.

🌟 Key Visual Elements
• Color Palette (Teal Theme):
  ◦ Primary: #00897B (Teal) - Memberikan kesan sehat, tenang, dan profesional.
  ◦ Background: #F4F6F8 (Soft Gray) - Mengurangi kelelahan mata saat chatting lama.
  ◦ Accent: #FFB300 (Amber) - Digunakan untuk elemen penting (seperti badge atau notifikasi).
  • Typography: Menggunakan font sistem Android yang modern dengan skala prioritas teks (Title, Subtitle, Body) untuk hierarki informasi yang jelas.

🤖 Chatbot UI Experience (Coach AI)
Interface chatbot dirancang menyerupai aplikasi messaging premium:
• Dynamic Chat Bubbles:
  ◦ Bubble User dengan warna Teal solid.
  ◦ Bubble Bot dengan warna putih bersih dan bayangan lembut (elevation).
• AI-Powered Badge: Lencana khusus pada header untuk memberikan identitas bahwa asisten didukung oleh AI.
• Quick Question Chips: Tombol interaktif di atas input field untuk memudahkan user bertanya tanpa mengetik (misal: Workout Plan, Healthy Diet, Track Progress).
• Smooth Visuals:
  ◦ Status "Online" dengan indikator titik hijau neon.
  ◦ Avatar robot yang minimalis dan ikonik.
  ◦ Input field melengkung (Rounded CardView) dengan tombol kirim yang mencolok.

  📱 Components & Resources
• Material Design: Implementasi CardView untuk elevasi dan sudut melengkung (rounded corners) pada setiap komponen.
• Custom Drawables: Penggunaan XML drawable untuk background chat yang adaptif terhadap mode terang/gelap.
• Emoji-Rich Interface: Penggunaan emoji yang kontekstual untuk membuat pengalaman aplikasi terasa lebih personal dan memotivasi.

  📱 Tech Stack
• Networking: OkHttp3 untuk Groq API.
• UI Framework: Android Material Components.
• AI Engine: Groq SDK (Llama 3.1 Model).
